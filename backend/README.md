# Basit Flask Backend

## Kurulum

1. Ortamı oluşturun ve bağımlılıkları yükleyin:
   
   ```sh
   python3 -m venv .venv
   source .venv/bin/activate
   pip install -r requirements.txt
   ```

2. Sunucuyu başlatın:
   
   ```sh
   python app.py
   ```

## Kullanım

- Sunucu çalıştığında `http://127.0.0.1:5000/` adresine giderek test edebilirsiniz.
